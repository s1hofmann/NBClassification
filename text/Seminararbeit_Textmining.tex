% 		scale - Skaliert die Grafik anhand eines Skalierungsfaktors. 

% 		height - Skaliert die Grafik auf die angegebene Höhe. Die Angabe der Größe muss in einer LaTeX-spezifischen 	%				 Einheit  erfolgen.
